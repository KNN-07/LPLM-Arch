# LPLM-1T-EP01
LPLM-1T-EP01 is a large language model with 1 trillion parameters, architecture adapted from DeepSeek V3 and Kimi K2 Series
Experiemntal model, not for production use. Please use with caution.
Training on Hugging Face open source pretraining datasets, filtered from CommonCrawl.

Note: LPLM-0.3B-EP01 is a smaller version of LPLM-1T-EP01, with 300 million parameters, for testing and evaluation purposes.
